#  Student ERP System

##  Overview

The **Student ERP System** is a web-based application designed to manage student-related data efficiently. It helps in handling records like student details, attendance, marks, and fees in one centralized system.

---

##  Features

*  Student Registration & Management
*  Attendance Tracking / Notice Board
*  Exam Form / Alert
*  Creat Class and Manage Class
*  Login-Logout
*  User: Admin, Teacher, Student
*  Secure Admin Access
*  Marks/Result Management
*  Fee Management
*  Search & Filter Students
*  Responsive Design


---

##  Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Django / Python
* Database: SQLite
